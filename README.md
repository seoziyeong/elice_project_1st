## 서비스 구성 안내

## 1. 서비스 소개

- HTML, CSS, JavaScript, React
- Node, Express, MongoDB, AWS S3

- 개요 : React와 Express 기반의 동적 페이지로 구성한 쇼핑몰 사이트


## 2. 서비스 주요 기능 설명

  - 상품, 장바구니, 주문, 사용자 및 관리자 권한 기능

## 5. 실행 방법
- Server : 
  ```bash
  1. mongodb (아틀라스)
  2. npm start
  ```
- Client :
  ```bash
  1. npm start
  ```
